• Select 组件

    《Select v-model="model" style="width:200px"》
    《Option v-for="item in statusTypelist" :value="item.value" :key="item.value"》{{ item.label }}《/Option》
《/Select》
    model:'0',
                statusTypelist:[
    {
        value: '0',
        label: '全部'
    }
]

    这里的双向绑定model，想要默认显示，model的默认值就必须跟statusTypelist里面的value值类型一样，是字符串，model也必须是字符串

  • 时间选择器

    《DatePicker type="daterange" @on-change="days" format="yyyy-MM-dd" placement="bottom" placeholder="Select date" 》《/DatePicker>》

    注：on-change 日期发生变化时触发 返回两个值，已经格式化后的日期，比如 2016-01-01，和当前的日期类型，比如 date

    所有注释里面的方法在html中写的时候都要加 ‘@’，所有的返回值都是直接在方法里面接收，不用行内传值，例子如下：

    days(res){
    console.log(res);  // '2016-01-01';
}
