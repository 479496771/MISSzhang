• 解决移动端弹框底层滑动

    document.ontouchstart = function (){
    return false;
};

    禁止移动端的触摸事件，一般在弹框中可以用，用来解决底层滑动

    用vue的话，可以在弹框最外层加 @touchmove.prevent,该方法只适用于弹框内部不需要滑动的情况（移动端）

    当弹框内部含有滚动的时候，可以在弹框出现时候给body加over-flow：hide,position:fixed;height:100%/100vh,（有个bug就是body会回到顶部，需要js获取当前页面的高，给body设置top值，在设定值时候还有个bug,就是获取不到页面高度，永远为0，只能获取页面相对滑动了的高度，）
